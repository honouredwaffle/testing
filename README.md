# 🧪 Game Testing Lab

A repo for experimenting with game designs — prototypes, mechanics, art tests, whatever. Nothing here needs to be finished or polished. Just ideas that need to be played with.

## Structure

```
/experiments    → individual game prototypes, one folder per idea
/assets         → shared sprites, sounds, fonts
/tools          → scripts or helpers for prototyping
```

## Live

Deployed via Vercel: [testing-qdip.vercel.app](https://testing-qdip.vercel.app)

## Rules

- Break things on purpose
- No pressure to finish
- If it's fun, it stays
